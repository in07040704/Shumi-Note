# Shumi-Note

## 制作方針

* 勉強内容を[notebooks](notebooks/)にまとめてます。
* 全てのnotebookは独立させています。

---

## インストール

poetryを使ってください。

```bash
git clone git@github.com:[your-repository]/Shumi-Note.git && cd Shumi-Note
poetry install
```
